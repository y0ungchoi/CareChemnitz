## How to use?

1. Creating .env file for the google maps API
   ```bash
   VITE_MAPS_API_KEY={your google maps API}
   ```
2. Running the Client:
   ```bash
   cd client && npm i && npm run dev
   ```
3. Running the Server:
   ```bash
   cd server && npm i && node --env-file=../.env server
   ```
